# PONK
A retro arcade game player made for Undercity by Estella and Froppii!!
Plays 3 games (pong, breakout, draw) with two analog sliders that people(up to 2!) can play with.

## Bill of Materials
- 2x analog sliders
- 1x orpheus pico
- 1x 2'8 TFT display
- cardboard
- wires
